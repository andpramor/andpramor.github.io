Sobre mí

En 2024, terminé con Matrícula de Honor el Grado Superior en Desarrollo de Aplicaciones Web, en el IES Hermanos Machado. Durante mis prácticas en Eviden, del Grupo Atos, adquirí una valiosa experiencia que me llevó a formar parte de su equipo como Desarrollador de Aplicaciones, rol que desempeño actualmente.

Mi enfoque principal es el frontend, donde estoy muy cómodo trabajando con ReactJS y Angular, y tengo sólidos conocimientos en HTML, CSS, Bootstrap, Javascript y Typescript.

En backend, he utilizado principalmente Python, con el framework web Django y DRF para desarrollar mis APIs REST, y también he realizado proyectos utilizando Java y Spring Boot. Tengo experiencia con varias bases de datos relacionales, en especial MySQL.

Y en cuanto a despliegues, empleo distintos servicios de Amazon Web Services (AWS) como buckets S3 para páginas estáticas y archivos, instancias EC2 para proyectos Django que necesitan ejecutar código y bases de datos RDS, además de GitHub Pages para proyectos estáticos cuando el proyecto no necesita otros servicios de AWS.
